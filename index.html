il2fb-events-parser demo

This is a demo page of il2fb-events-parser library.

The library is used by IL-2 Horus Team for parsing events log produced by dedicated server of «IL-2 Forgotten Battles».

See the list of events supported by the latest version of library.

This page is used for checking the library's ability to parse events from server's log file. Also, it shows parsing result.

If something goes wrong during parsing, then you will be asked to confirm automatical bug report (see open issues).
